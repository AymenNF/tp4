# Exécuter les différentes sortes de tests séparément

On ne veut pas toujours rouler tous les tests. 

1. Trouver une façon d'exécuter les tests unitaires, les tests composant et les tests multi-composants (Cucumber) séparément.
2. Inscrivez les commandes Maven dans le [fichier de questions](../questions/question-execution.md)

IMPORTANT: Pour faciliter la correction, `mvn test` exécuté à la racine du projet doit rouler tous les types de tests. 
**Une note de 0 sera donnée si les tests ne s'exécutent pas ou ne passent pas.**

ATTENTION: Tester vos commandes dans le terminal à la racine du projet. Ces commandes seront utilisées pour la correction.
**Une commande n'exécutant pas de tests donnera une note de 0 pour ce type de tests.**
